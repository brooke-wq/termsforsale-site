GHL Webhook Test Tool

Fire test payloads to your GHL webhooks so they detect all fields and let you save your workflows. Do this for each workflow before saving in GHL.

How to use this

1
In GHL go to Automation → Workflows → + New Workflow → Start from Scratch
2
Click + Add Trigger → Inbound Webhook — copy the webhook URL it shows you
3
Paste it in the field below and click Send Test
4
Go back to GHL — it will now show all the fields. Click Save, then add your actions (update contact, move to pipeline stage, notify, etc.)
5
Repeat for both workflows
